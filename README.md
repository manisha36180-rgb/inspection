# Vessel Inspection Report System

## Project Structure
- **/frontend**: Next.js 16 (Turbopack) frontend application.
- **/backend**: Supabase configuration and legacy backend logic.
- **/supabase**: Database migrations and Edge Functions.

## Getting Started

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`

### Features
- 51 Inspection Modules (Deck, Engine, Cargo, etc.)
- Multi-role Authentication (Admin, Superintendent, User)
- Real-time cloud sync with Supabase
- Interactive Inspection Table with persistence
- Admin User Management (Shanthi)
